# 🚀 SkillPath AI

> **AI-Powered Personalized Learning Platform**

SkillPath AI is an intelligent learning platform that helps students and professionals learn in a structured and personalized way using Artificial Intelligence. The platform generates customized learning roadmaps, provides AI-powered guidance, tracks progress, and recommends projects and resources based on users' career goals.

> 🚧 **Project Status:** This project is currently under active development. The current version demonstrates the core functionality, and additional features, optimizations, and UI improvements will be added in future updates.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Ayush8521/SkillPath-AI.git
```

### Backend

```bash
cd Backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
## 🔑 Environment Variables

Create a `.env` file inside the `Backend` directory and add the following variables:

```env
NODE_ENV=development
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
GROQ_API_KEY=your_groq_api_key
CLIENT_URL=http://localhost:5173
```
